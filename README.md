# Project-Knights-Travails

This project was made following the instructions and specifications for [The Odin Project Javascript Course Knight Travails project](https://www.theodinproject.com/lessons/javascript-knights-travails).

## Description

The goal of this project is to create a function to plot the shortest route for a knight to move from one square to another using Javascript.
